Requirements

This project was built using Python 1.2.1 along with Flask for the web interface.

To run this project locally, make sure you have the following installed:

pip install -r requirements.txt


If you don’t already have the file, you can generate one with:

pip freeze > requirements.txt



Core Dependencies

Python 1.2.1

Flask — for deploying the web app

scikit-learn — for building and training the Random Forest model

pandas — for data preprocessing

numpy — for numerical operations

matplotlib — for visualization (optional)

pickle — for saving and loading trained models

These dependencies ensure consistent performance for both model training and deployment.
